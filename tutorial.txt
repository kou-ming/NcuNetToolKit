宿網疑難雜症
優質文章：宿舍網路搞不定？ https://www.dcard.tw/f/ncu/p/234412414


1. 如何連上宿網
開學期前一個月大約會有1個月免費試用，只要註冊帳號就能用
結束後就需要繳錢，宿網每學期為800元，可用ATM轉帳或是超商付款，用ATM轉帳會比較快開通，快蠻多的

2. 如何註冊資料
宿網網址https://latias.cc.ncu.edu.tw/dormnet/index.php
登入後到 使用者專區 -> 申請/修改網路
接著就可以註冊網路，填入宿舍基本資料，接著要填入Mac Address

3. Mac Address查詢
Mac Address簡單來說是用來識別機器的，你的機器的Mac Address有填上去，這台機器才可以連上這個帳號開通的網路
-> 若是使用網路線直接接到自己的電腦
    1. 使用查詢Mac Address功能
    2. 若找不到可參考 https://www.koc.com.tw/archives/310582

-> 使用分享器：
    1. 電腦連上分享器
    2. 進入分享器的設定頁面
    3. 要選擇浮動IP
    4. Mac Address使用預設值就好
    5. 將這個Mac Address填入註冊資料

4. 使用注意事項
    1. 24小時內上傳量為3GB，超過會被限流
    2. 可以去宿網網站查詢流量，或是使用我們的流量警告功能
    3. 在早上6:00~7:30為早鳥優惠時段，那段時間不會計算上傳量

-----------------------------------------------------

4. 流量查詢
網管/記中公告 -> 流量查詢 -> 輸入IP位置即可查詢

5. 被超量限流但又剛好需要網路怎麼辦?
使用者專區 -> 急用申請
    1. 可以臨時解除限制，但期間所有流量仍列入「最近24小時內校內外上傳量」
    2. 每月可解除時間為12個月

6. 開通後還能退費嗎
退費申請表 https://latias.cc.ncu.edu.tw/dormnet/dormnet_refund.pdf
